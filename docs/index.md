# Feedback

A modern TUI podcast client for the terminal.

## Features

- Subscribe to RSS/Atom podcast feeds
- Stream or download episodes
- Playback with VLC or MPV
- Progress tracking and resume
- Queue management
- OPML import/export
- Fully customizable keybindings
- Beautiful terminal UI built with [Textual](https://textual.textualize.io/)

## Quick Start

```bash
pip install feedback
feedback
```
